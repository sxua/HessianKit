HessianKit

HessianKit is a Framework for Objective-C 2.0 to allow OS X and iOS applications to seamlessly communicate with hessian web services.

HessianKit v.2.0.0

HessianKit is a Framework for Objective-C 2.0 to allow OS X 10.5+, and iOS 4.0+ applications to seamlessly communicate with hessian web services. Hessian web services can be implemented using Java, .NET and more.

HessianKit provided typed proxy objects to formward calls to, and handle responses from any hessian web service, seamlessly just as if they where local objects. HessianKit also provide functionality to automatically generate classes for published value objects. Both web service proxies and value objects are defined as basic Objective-C Protocols.

Installation

The easiest way to get HessianKit installed is to use CocoaPods. Just add this line to your Podfile:

pod "HessianKit", "~> 2.0.0"

Documentation

Documentation can be found at CocoaDocs. Also it can be generated using an appledoc or a native utility headerdoc2html.

Contents

The HessianKit project consist of several targets:

  • Documentation generates headerdoc documentation for the OS X and iOS APIs.
  • HessianKit generates a framework to be bundled with applications for Mac OS X 10.5 and later, includes documentation.
  • StaticHessianKit generates a static library to be linked with applications for iOS 4.0 and later.
  • MacTests executes unit tests for HessianKit target against Mac OS X 10.5 and later.
  • iPhoneTests executes unit tests for StaticHessianKit against iOS 4.0 and later.
